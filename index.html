Features

Everything you need.
Nothing you don't.

A focused set of tools built for Safaricom agents who need reliability and speed above all else.

USSD Automation
Intercepts M-Pesa SMS and dials USSD automatically — no screen unlock, no manual input, 24/7.
Multi-Step USSD
Full multi-step flows with per-step responses, conditional navigation, and smart detection.
Fallback USSD
Switches automatically to a backup USSD code when the primary flow fails. Zero intervention.
SMS Relay
Forward M-Pesa messages to partner agents. Sender and receiver modes run simultaneously.
Web Orders
Your own order page. Customers pay via STK push, app fulfils instantly.
Scheduler
Schedule transactions for a future time with configurable retry windows on failure.
Agent Alerts
Real-time SMS alerts for automation-off events, failures, reversals, and blacklisted numbers.
Dual SIM
Assign offers to specific SIM slots. Run different services on different SIMs simultaneously.
Transaction Logs
Full searchable history — status, USSD response, failure reason, original SMS.
Blacklist
Block specific customers, disable offers individually, prevent duplicate executions.
Offers Bot
SMS chatbot that recommends offers, answers inquiries, and collects recipient numbers.
Auto-Save Contacts
Saves paying customers with a configurable suffix. Handles duplicates intelligently.
How It Works

Four steps.
Fully automatic.

01
Customer Pays
M-Pesa payment arrives on your till. App reads the SMS instantly — no delays.
02
Offer Matched
Amount matched against your configured offers. Correct USSD selected automatically.
03
USSD Dialled
Accessibility service executes the USSD sequence silently. Screen stays off.
04
Done & Logged
Result saved with full USSD response. Automatic retry on failure — or flagged for review.
